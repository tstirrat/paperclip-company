# Agent Company

> Built from the [agentsys](https://github.com/agent-sh/agentsys) template by [Paperclip AI](https://paperclip.ing)

An [Agent Company](https://agentcompanies.io) package — an AI-powered software engineering company that orchestrates agents across the full development lifecycle.

## Org Chart

```
CEO
├── CTO
│   ├── Staff Engineer
│   └── Research & Perf Analyst
├── QA & Release Lead
└── Heartbeat Dispatcher
```

The Heartbeat Dispatcher runs on a timer to scan for stalled work. All other agents wake on assignment or mention.

## Getting Started

```bash
npx paperclipai company import this-github-url-or-folder
```
